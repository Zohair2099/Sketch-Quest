# **App Name**: SketchQuest

## Core Features:

- User Authentication: Secure user login and signup with role-based access (Student, Teacher, Admin).
- Interactive Quests: Quest-based lessons with mini-quizzes (MCQs, drag-drop) and XP rewards.
- Performance Tracking: Dashboards for students and teachers to track progress, XP, badges, and streaks.
- AI Lesson Recommendations: AI tool to analyze performance and recommend next best lessons. LLM tool evaluates student's previous exercises to make recommendations.
- Leaderboard: Rank students (class, school, global) with XP, rewards and coins.
- Teacher Dashboard: Tools for teachers to manage students, assign quests, and view analytics.
- Push Notifications: Send streak reminders and daily learning reminders using push notifications.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke a sense of trust and learning.
- Background color: Light blue (#E0F7FA) to provide a calming and student-friendly backdrop.
- Accent color: Yellow (#FFDA63) to highlight interactive elements and CTAs.
- Body and headline font: 'PT Sans', a modern humanist sans-serif, ensures readability and contemporary appeal.
- Gamified icons: Use playful, rounded icons related to quests, XP, streaks, and power-ups.
- Rounded cards: Apply soft shadows on cards for depth and a user-friendly feel.
- Animated progress bars: Use dynamic progress bars and animations for XP gains and streaks.
